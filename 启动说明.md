# Todo 系统启动指南

本项目包含后端 (FastAPI) 和前端 (Next.js) 两个部分，请按照以下步骤启动。

## 1. 准备工作
确保你的本地 **MySQL** 已经启动，且配置如下：
- **数据库名**: `todo_db`
- **用户名**: `root`
- **密码**: `root`

> 如果你是第一次运行，请先执行数据库初始化脚本：
> ```powershell
> python init_db.py
> ```

---

## 2. 启动后端 (API 服务)
后端负责处理数据存储和业务逻辑。

1. 打开一个 **新的终端**。
2. 确保位于项目根目录 `my-project`。
3. 执行启动命令：
   ```powershell
   python -m uvicorn src.backend.main:app --reload
   ```
4. **验证**: 看到 `Uvicorn running on http://127.0.0.1:8000` 表示启动成功。
   - API 文档地址: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

---

## 3. 启动前端 (Web 界面)
前端是用户操作的网页界面。

1. 再打开一个 **新的终端**。
2. 进入前端目录：
   ```powershell
   cd src/frontend
   ```
3. 执行启动命令（如果直接运行 `npm` 报错，请使用 `.cmd` 版本）：
   ```powershell
   npm.cmd run dev
   ```
4. **验证**: 看到 `ready - started server on 0.0.0.0:3000` 表示启动成功。
5. **访问**: 在浏览器打开 [http://localhost:3000](http://localhost:3000)

---

## 4. 常见问题处理
- **PowerShell 脚本限制**: 如果报错“禁止执行脚本”，请在执行命令前运行：
  ```powershell
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
  ```
- **端口冲突**: 如果 3000 或 8000 端口被占用，请关闭相关程序后重试。
- **数据库连接失败**: 请检查 `src/backend/core/database.py` 中的 MySQL 配置是否与你本地一致。
